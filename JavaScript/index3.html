La Boucle while

La boucle while va nous permettre de répeter une série d'action tant qu'une condition est vraie.

La boucle do..while

La boucle for

Elle nous permet de répeter une série d'instructions à partir d'une valeur donnée jusqu'à une certaine valeur seuil.

La fonction

C'est un bloc de code qui va éxecuter une tâche précise
